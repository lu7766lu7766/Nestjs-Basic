## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
AppTask basic api project

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migration

```bash
# watch entitys to generate diffirent or new schema code
$ npm run migration:generate

# run migration in new version
$ npm run migration:run

# rollback migration
$ npm run migration:rollback
```

## Seed

```bash
# run config
$ npm run seed:config

# run seeds
$ npm run seed:run
```
